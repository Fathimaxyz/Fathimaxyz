## Featured Projects

### 🤖 Interview AI — AI-Powered Career Assistant
> Full-Stack · React.js · Node.js · Express · MongoDB · Gemini API · JWT · Puppeteer · Vercel

An end-to-end AI career assistant that parses resumes with Google Gemini API to generate structured skill gap reports with 5+ actionable insights per analysis. Features JWT authentication with server-side token blacklisting, dynamic ATS-optimized PDF generation via Puppeteer (under 3s), and is deployed on Vercel + Render with global CDN.

### 🥗 AI Pantry & Recipe Manager
> Full-Stack · React.js · Node.js · Express · PostgreSQL · Neon · Gemini API · Tailwind CSS v4

A full-stack pantry management system built on the PERN stack. Integrates Gemini 2.5 Flash to generate personalized recipes from live inventory in under 5 seconds, supporting 5+ dietary filters. Designed across 6 normalized PostgreSQL tables hosted on Neon with sub-200ms indexed queries. Features automated expiry alerts, a 7-day meal planner, and bcryptjs-secured authentication.

[![GitHub](https://img.shields.io/badge/GitHub-181717?style=for-the-badge&logo=github&logoColor=white)](https://github.com/Fathimaxyz/ai-recipe-generator.git)
